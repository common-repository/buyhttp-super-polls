Easily create and manage polls for your site. Then display results in a post or page.

== Description ==
Use this plugin to create and manage polls you can put anywhere on your site with a widget. Features:

- Widget for placing polls
- [bsp_show_results] shortcode for displaying poll results
- Easy administration of new polls and options
- Fully translatable

== Installation ==
- Install and activate plugin
- Go to Pages > Add New and create a new page with the [bsp_show_results] shortcode
- Go to Settings > BSP Settings and enter the ID for the page you created
- Go to Plugins > Manage BSP and create your first poll
